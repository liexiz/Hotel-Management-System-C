Hotel Management System

C project for managing hotel rooms, user accounts, and bookings using linked lists and file handling.

Features
- User registration and login
- Admin and user menus
- Room management
- Booking creation and cancellation
- View available rooms
- View user bookings
- Account management
- File saving and loading
- Basic password encryption
- Dynamic memory management

Room Types
- Standard
- Deluxe
- Suite
